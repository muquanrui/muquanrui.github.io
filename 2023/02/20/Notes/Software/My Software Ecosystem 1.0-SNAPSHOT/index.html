Indispensable

Routine

Apple Calendar：日程

  • 记录每日日程
  • 设置确定时间的日程提醒

Apple Reminders：任务

  • 展示并提醒每日目标任务
  • 设置待办事项、周期事件、学习任务、工作目标等清单

Notion：记录

  • 记录各种复杂且信息多维度的列表或数据库
  • 记录长期目标、短期目标、周期目标完成情况
  • 体系化、主题化且少有更改的文字内容
  • 临时性、落笔则废、阅后即焚的简单文本
  • 汇总并分类各种文本、数据库

有鱼记账：记账

  • 消费记录
  • 账户查看
  • 金钱管理

Practice

Typora：文字

  • 关于专业知识、技术学习、项目经验、职业发展的笔记、记录、总结、文章
  • 系统性、体系化的学习笔记、读书笔记
  • 模块化、有格式要求、内容长的文字工作
  • 时常查看、时常更新的主题内容
  • Markdown编辑器，Blog写作工具

XMind：结构

  • 逻辑化、分支多的主题结构
  • 思维导图化的过程、目标、记录
