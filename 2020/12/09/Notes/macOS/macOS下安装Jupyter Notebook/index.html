macOS下安装Jupyter Notebook

  • macOS
  • 2020-12-09
文章字数:248字     阅读时长:1分钟

因为这学期的C语言课程老师的课本使用的是Jupyter Notebook，因此需要这个基于Web的交互式笔记软件来进行这门课程的学习，于是查找了一些教程尝试进行安装。

因为jupyter是一个python库，可以利用pip来进行安装。

如果mac上安装了python3版本，打开终端输入命令pip3 install jupyter。

然后使用命令jupyter notebook进行启动。

会自动使用默认浏览器打开Jupyter Notebook的主页。

但是现在的笔记本只有python3的内核，还需要手动安装C内核，才能新建可运行C语言的笔记本。

保持Jupyter Notebook的打开，在终端使用下面的命令：

pip3 install jupyter-c-kernel
install_c_kernel

可以看到C内核已经成功安装了。

把下载下来的笔记本挪到一个目录下，从Jupyter的目录树下进入到存放笔记本的目录，双击打开就大功告成了。
