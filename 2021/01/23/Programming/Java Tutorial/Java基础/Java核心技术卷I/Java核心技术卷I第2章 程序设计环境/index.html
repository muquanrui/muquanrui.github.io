2.1 Java开发工具包

JDK - Java Development Kit：编写Java程序的程序员使用的软件
JRE - Java Runtime Environment：运行Java程序的用户使用的软件

JDK是Java的开发工具包。JDK主要包含三部分，第一部分是Java运行时环境JRE，核心内容JVM（Java虚拟机）；第二部分是Java的基础类库；第三部分是Java的开发工具，包括编译、运行工具javac、java等。
JRE只是Java程序的运行环境，核心内容是JVM和运行时类库。

简而言之，编写Java程序需要JDK，运行Java程序只需要JRE。

安装过程见macOS下Java开发环境搭建。

2.2 命令行工具

命令行编译运行Java程序的步骤：

  1. javac HelloWorld.java
  2. java HelloWorld

javac是一个Java编译器，它将java源文件编译成class文件，java启动JVM，虚拟机执行class文件中的字节码。
